# Personal Auditor REST API

## Specification
- MongoDB
- mongoosejs
- express
- body-parser

### Requirement: 
- nodemon

### Install nodemon
`npm install -g nodemon`

### Clone this repo
`git clone https://github.com/afifkhaidir/personal-auditor-api.git`

### Go to project dir
`cd personal-auditor-api`

### Run the server
`nodemon index.js`
